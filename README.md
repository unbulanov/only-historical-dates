# Historical-Dates

# Используемые технологии:
- Typescript
- Webpack
- React
- SCSS
- GSAP для анимаций
- Swiper

## Запуск проекта локально:

```
git clone git@github.com:unbulanov/only-historical-dates.git
```

```console
npm run build:dev
npm run build
npm run serve
```
```
Откройте браузер и перейдите по адресу http://localhost:8080/, чтобы просмотреть проект.
```
